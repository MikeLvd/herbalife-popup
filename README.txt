Advanced responsive popup notice for Herbalife sites with modern UI and enhanced controls.

== Description ==

Herbalife Popup Notice is a modern, secure, and feature-rich WordPress plugin designed specifically for Herbalife distributors. It displays a professional popup notice to visitors with customizable content, appearance, and behavior.

= Features =

* **Modern Architecture**: Built with PHP 8.0+ standards, namespaces, and OOP principles
* **Enhanced Security**: Implements WordPress security best practices
* **Responsive Design**: Mobile-first approach with dark mode support
* **Multiple Triggers**: Time delay, scroll position, or exit intent
* **Smart Display Rules**: Control where and when the popup appears
* **Cookie Management**: Remember user preferences with configurable duration
* **Accessibility**: WCAG 2.1 compliant with proper ARIA labels and keyboard navigation
* **Performance Optimized**: Lazy loading and efficient resource management
* **Analytics Ready**: Built-in tracking for popup interactions
* **Translation Ready**: Full i18n support

= Requirements =

* WordPress 5.8 or higher
* PHP 8.0 or higher
* Modern browser with JavaScript enabled

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/herbalife-popup` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Settings > Herbalife Popup to configure the plugin
4. Customize your popup content, appearance, and behavior
5. Enable the popup when ready

== Frequently Asked Questions ==

= Can I use this plugin on non-Herbalife sites? =

Yes, while designed for Herbalife distributors, the plugin can be customized for any website requiring a professional popup notice.

= Does it work with WooCommerce? =

Yes, the plugin includes special integration for WooCommerce cart and checkout pages.

= Is the popup accessible? =

Yes, the popup follows WCAG 2.1 guidelines with proper ARIA labels, keyboard navigation, and focus management.

= Can I track popup performance? =

Yes, the plugin includes built-in analytics tracking that can be extended with your preferred analytics platform.
